Matrix Bot — Privacy Policy

Last Updated: October 2025

1. Introduction

Matrix Bot (“we”, “our”, or “the bot”) is a Discord application designed to help communities manage esports tournaments, scrims, roles, and moderation tools. We respect your privacy and are committed to collecting only the minimum data required for normal bot operation.

2. Information We Collect

Matrix Bot collects limited, non-personal data necessary to function within Discord. This may include:

  • Discord user IDs (for tournament registration, moderation actions, and role assignment)
  • Guild (server) IDs and channel IDs (for configuration and tournament data)
  • Team names, slot numbers, and tournament-related settings

Matrix Bot does NOT collect or store:

  • Message content
  • Emails, passwords, payment information, or real-world personal data
  • Private messages (DMs)

Message content may be temporarily processed in real-time for moderation purposes, but it is never stored, logged, or saved.

3. How We Use the Data

Collected data is used only within Discord to:

  • Manage tournaments, scrims, and team registrations
  • Assign or remove roles automatically
  • Create and manage private group channels and permissions
  • Apply moderation actions and server protections
  • Generate tournament tables or visual summaries displayed in Discord

We do not sell, share, or use collected data for advertising, analytics, or tracking purposes.

4. Data Storage & Security

Only essential configuration and tournament data (IDs and settings) are stored on the bot’s hosting server. Access is restricted to authorized administrators. Reasonable security measures are used to protect stored data. No system is 100% secure.

5. Data Retention & Deletion

  • Tournament data is retained only while an event is active
  • Data may be automatically deleted after a tournament ends
  • Server owners may request a full data wipe via bot commands or support

Removing the bot from a server deletes all associated server data.

6. User Control & Opt-Out

Users and server owners may opt out of data usage by:

  • Disabling specific bot features via available commands
  • Requesting data deletion
  • Removing the bot from their server

7. Third-Party Services

Matrix Bot interacts only with Discord’s official API. No third-party analytics, tracking, or advertising services are used.

8. Your Rights

  • Request access to stored data
  • Request deletion of your data
  • Remove the bot from your server at any time
  • Contact us for clarification regarding data handling
